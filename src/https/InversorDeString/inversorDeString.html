Inversor de String

Uma String é uma sequência de caracteres sucessivos, usados para representar palavras, frases, ou textos de um programa.
Na programação é possível manipula-las da maneira que desejarmos a partir de arranjos de bytes.

E é isso que esse programa faz, pega um texto ou palavra e manipula o arranjo para inverter a String.

Exemplo: Target Sistemas passa a ser: sametsiS tegraT
